# z-spanish 需求 v2（2026-07-02 讨论整合 · 开发基准）

本文件整合 2026-07-02 讨论的全部决定，**冲突时以本文件为准**，正式覆盖 `z-spanish-原始需求.md` 与 `z-spanish-调整补充-v1.md` 的相应条目。

## 一、定位（更新）

把**马德里日常真实会用到的地基级西语**，拆成**词块**，用**主动产出**的方式，练到能**自己拼出简单正确的句子**。

- 唯一方向：中文意图 → 说 / 写西语。
- **明确不做**：理解 / 听力 / 西语→中文 / 语音识别 / 音频（本轮一度考虑理解与音频，已取消，回到"主动输出优先"）。

## 二、本轮相对旧文档的正式变更

1. **【新增】训练从"回忆固定块"扩展到"自己拼对"**：冠词(el/la、un/una)、ser/estar、阴阳性数一致、人称变位等"拼句零件"要能主动产出。
   → **推翻** 调整补充 v1 第二节（"不单独强调阴阳性 / 冠词整体记忆"）。现在冠词与性数是**要主动练的技能点**。
2. **【新增】分类改为双轴**：场景轴 + 语法点轴，两轴都能发起"分类练习"。解决旧"必背 20 类"把语法轴和主题轴混用导致的混乱。
3. **【新增】内容标准 = 马德里本地日常真实说法**。硬性排除：课本化假句、游客化说法、用不上的词表。
4. **【取消】理解 / 听力 / 西语→中文 / 语音识别**：保持产出单一方向。
5. **【保留不变】**：词块不孤立、间隔复习节奏(今天/2/4/7/14/30/60/90)、每日 20(新≤5)、"我不会"进学习模式、静默计时兜底、训练页不显示熟练度、无 streak/百分比、宁少勿错、App 外生成→人工审核→批量导入。

## 三、内容分两层

- **地基层（通用，所有场景的底座）**：礼貌社交套语 / 自我·他人信息 / 存在与位置 / 想要·需要·能·得 / 数字·钱·时间 / 拼句零件(语法点)。
- **场景层（真实生活，按频率排序）**：
  - S1 早餐店·咖啡馆 → S2 小酒吧 → S3 超市 → S4 买衣服 → S5 看病·前台挂号 → S6 看病·跟医生 →（贯穿）S7 陪父母（替父母说 / 说明父母情况）。

## 四、双轴分类

### 场景轴（`category`）
地基层各模块 + S1–S7。

### 语法点轴（`grammar_point` / `tags`）
- G1 定冠词 el/la（含 el agua 特例）
- G2 不定冠词 un/una
- G3 ser / estar
- G4 hay / está
- G5 人称变位（高频动词）
- G6 性数一致（una camiseta blanca）
- G7 介词搭配（para/por、a/en、con）
- G8 反身动词（me llamo、me duele、me lo pruebo）
- G9 与格 gustar/doler（me gusta、me duele）
- G10 礼貌软化式（quería、podría）

每个词块：`category` = 场景模块；`grammar_point`/`tags` = 它锻炼的语法点；`type_zh`/`subtype` = 角色（冠词块 / 搭配块 / 功能句）。

### 分类练习（新功能，带守护栏）
- 可按**场景**或**语法点**发起定向练习（例："最近在练冠词" → 抽所有 G1/G2 的块）。
- 是基础库里的**次要入口**，视觉明显弱于首页"开始背诵"。
- **不替代、不占用**每日 20 的排期；作答照常写回 SRS（练了不白练，也不制造影子进度）。

## 五、"会拼句"的训练含义

- 题目仍是中文意图 → 西语产出。
- 覆盖三种粒度：最小块（`una caña`）、搭配块（`me llevo esta`）、完整功能句（`¿Me pones un café con leche?`）。
- 一部分题的**考点就是"拼对零件"**：给中文，你要自己产出正确的冠词 / 性数 / 变位，而不是背一个固定串。
- **判定**：near_correct 宽松判定不变（只差重音 / 大小写 / 标点）。但**冠词错、性数错、变位错属于 wrong**——这是要练的核心技能，不能宽纵（当前判定逻辑天然满足：字母不同即 wrong，无需改）。

## 六、内容标准（马德里真实）细则

- 半岛西语，含 vosotros。
- 真实口语但**有界**：收 `¿me pones?` / `¿me cobras?` / `una caña` / `vale` / `buenas`；不收生僻俚语、粗话、地区差异过大的说法。
- 每条过双关：**马德里人真的这么说 ✓ + 我真的会用到 ✓**。
- `example_es` 用真实语境；`note` ≤ 1 句，只点关键区分，不展开语法课。

## 七、落地节奏

1. 本 v2 文档 + S1 样例内容审过 → 定稿。
2. 按新双轴调 schema（加 `grammar_point`；基础库加"按场景/语法点练"入口）。
3. 一个场景一个场景产出内容（S1→S2→…），每批人工审核后导入。
4. 旧的 49 条种子按新标准**重挑 / 重标**，不合格的淘汰。

## 八、样例：S1 早餐店·咖啡馆（待审）

| 中文意图 | 西语 | 语法点 | 备注 |
|---|---|---|---|
| 你好（进店随口） | Buenas. | 礼貌 | 马德里口语，比 buenos días 更随口 |
| 给我来杯咖啡（加奶） | ¿Me pones un café con leche? | G2, G5 | 真实点单用 ¿me pones?，不是 quiero |
| 还要一个可颂 | Y un cruasán también. | G2 | un + 阳性名词 |
| 一杯水 | Un vaso de agua. | G2 | agua 特例，但这里用 un vaso 规避 |
| 带走 | Para llevar. | G7 | |
| 在这儿喝 | Para tomar aquí. | G7 | 与 para llevar 对照 |
| 多少钱？ | ¿Cuánto es? | 数字钱 | 结账口语说法 |
| 麻烦买单（不急时） | Cuando puedas, ¿me cobras? | G5 | 马德里很自然的结账说法 |
| 能刷卡吗？ | ¿Con tarjeta? | G7 | 口语省略，比整句更真实 |
| 就这些，谢谢 | Nada más, gracias. | 礼貌 | 收尾 |
